# Geng_Lottery  
  
(Run on Linux)  
Thai lottery 3-digit top number analysis program

(ใช้บน ลีนุ๊กซ์ เท่านั้น)

# วิธีใช้  
เมื่อดาวน์โหลดมาแล้ว ให้คลิ๊กขวาที่โปรแกรม "geng_lottery" เลือก คุณสมบัติ -> สิทธิ์->  
ติ๊กเครื่องหมายถูกที่ "อนุญาติให้เลือกแฟ้มทำงานเป็นโปรแกรม" ตามรูปนี้

# _____________________  
  
จากนั้นเปิดโปรแกรมโดยการ ดับเบิ๊ลคลิ๊กไฟล์ "geng_lottery"  
1. ในโหมด ป้อนเอง ให้ใส่เลขอะไรก็ได้ 3 ตัวขึ้นไปที่คิดว่าน่าจะมาในงวดนี้เช่น 123456  
แล้วกดเริ่มวิเคราะห์ จากนั้นกด ออกรายงาน เพื่อรายงานผลเป็น Text ไฟล์
  
2. ในโหมด อัตโนมัติ แนะนำให้เปลี่ยนจาก 3 เป็น 6 แล้วกด เริ่มวิเคราะห์  
โปรแกรมจะดึงสถิติหวย 20 งวดล่าสุดมาคำนวนด้วยระบบ Ai แล้วรายงานผลออกมา  
ให้ซื้อตามนั้น

แนะนำ ซื้อตัวละ 1 บาท ทั้งหมด 120 ตัวก็เป็น 120 บาท  
ถ้าถูกเลขใดเลขหนึ่งใน 120 ตัวนี้ คุณก็จะได้ 900 บาท คือได้กำไร 780 บาท
